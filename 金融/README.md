## 使用说明

1. **文件命名规范**：使用`YYYY-MM-DD_分析主题.md`格式，如`2026-03-05_恒生科技指数分析.md`
2. **数据格式**：优先使用Markdown格式，便于阅读和版本控制
3. **数据引用**：引用外部数据时请注明来源和获取时间
4. **更新频率**：定期更新分析报告，保持信息时效性
5. **数据获取**：使用`数据源/`目录下的工具获取最新数据

## 工具与数据获取

### 数据获取系统
已建立完整的数据获取系统，位于`数据源/`目录：

#### 快速开始
```bash
# 1. 安装依赖
pip install -r 金融/数据源/requirements.txt

# 2. 运行数据获取
cd 金融/数据源
python scripts/fetch_financial_data.py

# 3. 选择获取选项
#   1 - 获取所有数据
#   2 - 仅获取价格数据
#   6 - 测试腾讯股票
```

#### 主要功能
- **多数据源支持**：雅虎财经、新浪财经、东方财富等
- **多种数据类型**：股票价格、财务报表、指数、宏观数据
- **自动化获取**：支持定时任务和批处理
- **完整日志**：记录所有操作和错误

### 推荐工具
- **数据获取**：Python (pandas, yfinance, akshare)、JavaScript (axios)
- **数据处理**：Excel、Python (numpy, pandas)、SQL数据库
- **可视化**：ECharts、D3.js、Python (matplotlib, seaborn, plotly)
- **报告撰写**：Markdown、Jupyter Notebook、Quarto
